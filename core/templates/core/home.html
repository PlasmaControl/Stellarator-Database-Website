About the Project

The Stellarator Database is a collaborative platform for the stellarator fusion research community. It provides a centralized repository of equilibrium simulation results computed with the DESC and VMEC codes, along with device configurations, publication references, and associated data files.

Researchers from participating universities and laboratories can upload their DESC equilibrium results, browse existing simulations, and query the database using flexible search criteria.

What's in the Database

  • desc_runs — DESC equilibrium simulation results with spectral resolution parameters, stability metrics, and profile data
  • vmec_runs — VMEC equilibrium simulation results
  • configurations — Stellarator magnetic configuration geometry (Fourier coefficients, profiles)
  • devices — Physical stellarator devices (W7-X, HSX, NCSX, etc.)
  • publications — Linked academic publications

Getting Started

  1. Register for an account and wait for admin approval.
  2. Use the stelladb Python package to prepare your DESC results for upload, or manually prepare a CSV export. See the GitHub repository for details.
  3. Navigate to Upload Data to submit your equilibrium files and CSV metadata.
  4. Use Search Data to query the database with SQL-style filters across any table.
  5. Check Variable Names for a full description of every database column.
